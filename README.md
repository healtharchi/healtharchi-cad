# HEALTH ARCHI CAD

> 小規模建設業・小規模建築のための、軽くて素直な 2D 製図ツール。
> インストール不要、HTMLファイル1枚で動く、業界標準 DXF 入出力対応。

## このツールが解決すること

- AutoCAD は機能過剰でサブスクが高い
- JW-CAD は操作性が独特で習得に時間がかかる
- でも「現場で図面を素早く確認・修正したい」だけなのに、選択肢がない

このツールは、**従業員2〜30名の工務店・一級建築士事務所が「ちょっと描く・確認する」のために必要な機能だけ**を、ブラウザで完結する形にまとめました。

## 想定ユーザー

- 図面を頻繁にやり取りする小規模工務店の現場監督
- 簡易な手書き図を清書したい大工・職人
- 出先で図面を素早く確認・修正したい一級建築士
- AutoCAD のサブスク費用を見直したい個人事業主

## 主な機能

- 線・矩形・円・楕円・ポリライン・配列・トリム・オフセット
- 寸法線・文字
- 画層（レイヤー）管理＋線種・色制御
- **DXF 形式の保存・読込**（AutoCAD / JW-CAD 互換）
- 図面枠・印刷（縮尺対応）
- 複数タブで複数図面の同時編集
- スナップ・グリッド・グリッドプリセット
- Undo / Redo
- サンプル図面付き

## 使い方

### 1. ダウンロードして開くだけ

1. `healtharchi-cad.html` をダウンロード
2. ダブルクリックでブラウザが開く（Chrome / Edge / Safari 推奨）
3. 起動画面の「新規図面」または「サンプル図面」をクリック

インストール不要、インターネット接続不要、データはブラウザ内だけに残ります（**当社サーバーには一切送信しません**）。

### 2. オンライン版（実験室）

ダウンロード不要で使いたい方は、HEALTH ARCHI 実験室から起動できます：
👉 https://healtharchi.com/tools
